🎯 How to Play

Setup

  • • Choose language, knowledge level and time per question.
  • • Pick or generate 6 categories (required for the game).
  • • Optionally add a theme to shape generated questions.

Invite players

  • • Create a room to get a 6-digit code and join link.
  • • Show the QR code on the main screen.
  • • Players join on their phones with the code or QR.

During questions

  • • Each question appears on players’ devices with up to 4 options.
  • • Use the timer controls to pause, resume or extend the time.
  • • You can regenerate a question or show the explanation when ready.
  • • Auto-advance can move to the next question after a short delay.

Scores & results

  • • Track live scores and answer status in the side panel.
  • • Final screen shows ranking based on the current scoring rules.
  • • Best experience: host view on a big screen, players answer on phones.
